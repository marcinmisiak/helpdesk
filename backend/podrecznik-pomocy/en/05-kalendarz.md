# Calendar

The **Calendar** view presents your tickets and deadlines in a monthly or weekly plan — making it easier to plan your work.

## How the Calendar Works

The calendar shows tickets with the following dates:
- **Completion deadline** — derived from the SLA agreement or set manually
- **Return date** — for deferred tickets

## Navigating the Calendar

| Action | Description |
|--------|-------------|
| **< Previous** | Go to the previous month/week |
| **Today** | Return to the current date |
| **Next >** | Go to the next month/week |

## Calendar Items

Each calendar event displays:
- The ticket number or subject
- A colour depending on priority or status

Click an event to go directly to the ticket details.

## Event Colours

- 🔵 **Blue** — tickets in progress
- 🟡 **Yellow** — new tickets waiting to be assigned
- 🔴 **Red** — overdue or high-priority tickets
- 🟢 **Green** — deferred tickets with a return date

## Tips

- Use the calendar to plan your working week
- Pay attention to clusters of tasks — they may indicate overload
- Red items require an immediate response
